使用手機掛網能賺錢！honeygain 教學

發表於 2020-02-21 | 分類於 程式

前言

　　這個網站名稱叫 honeygain，是一個網路賺錢的網站，但是這個網站：不用看影片、不用填問卷，什麼都不用做，只要「掛網」就好。沒錯！不要懷疑，就是只要「掛網」，掛著就能賺錢了！

honeygain 是什麼？


　　「honeygain」翻成中文的意思是「採集蜂蜜」。而其網站的運作機制的確是很像採集。
　　
　　這個網站會成立，其實是為了市場商業研究，藉由觀察使用者的「網路流量」，來獲取一些數據資料，並提供公司企業作參考決策。而使用者的獲利來源則為公司企業所提供的研究資金。
　　
　　目前honeygain網站提供三種版本：　　

  • Windows 電腦：Windows
  • MacOS 電腦：MacOS
  • 手機：Android系統

honeygain如何使用

　　(接下來的文章會以電腦版示範為主，電腦版與手機版功能都一樣，對照著操作不會太難，如果有問題歡迎來信提問。)

步驟1：註冊新帳號
　　首先我們點擊此連結。進入此網頁之後，如下圖所示。點擊紅色框框進行註冊。

　　點擊後，出現下圖，依照要求填入註冊資訊。在註冊完成之後，即會獲得5美元(如紅色框框所示)。

　　依照要求，填入

  • Email
  • 登入密碼

　　Email我是都以Google的gmail為主。輸入完成之後，點擊藍色框框，進入下一下個步驟。

步驟2：信箱認證
　　進入步驟1所填的Email收信，即可以看到honeygain寄來的認證信。如下圖所示。

　　進入信箱，點擊「Verify email」紅色框框，認證就完成了。

步驗3：安裝軟體

  • Windows 電腦
    　　點擊此「連結」，開始下載honeygain 軟體。下載安裝完成之後，開啟honeygain 即可以在電腦的右下角選單看到你的蜜蜂圖示，此時登入你剛剛申請的honeykgain帳號，即可以開始你的掛網之旅了。

    　　登入之後，即可以看到已經有5美元入帳。

  • Android 手機(Android 版本為9)
    　　以Nokia 3.1手機為例。我們需要使用Chrome下載App Apk檔案安裝，因此需要開啟App安裝權限。以下說明如何開啟安全性權限。
    　　選擇手機設定，並進入「應用程式和通知」。

    　　往下滑，選擇「特殊應用程式存取權」。

    　　選擇「安裝不明應用程式」。

    　　選擇「Chrome」。

    　　開啟「充許這個來源的應用程式」。

　　使用「Chrome」開啟此「連結」。選擇Android並開始下載。下載完成之後，點擊進行安裝。如果不能安裝請檢查「Chrome」的App安裝權限是否已經開啟。

　　安裝完成之後，點擊Honeygain開啟。點擊同意進入下一步。

　　點擊跳過介紹，進入下一步。

　　進入登入畫面，登入你剛剛申請的honeykgain帳號，進入下一步。

　　登入之後，「User mobile data」選擇「Yes」，「Allow usage on battery」選擇「Yes」，然後選「Done」。

　　完成之後，可以看到Dashboard上面顯示已經有5美元，現在開始可以使用你手機進行掛網之旅了

honeygain功能介紹

　　(honeygain的Dashboard，電腦版與手機版功能都一樣，對照著操作不會太難，因此這裡以電腦版操作為主，如果有問題歡迎來信提問。)

  • Dashboard 主畫面
  • 流量
    可以看到過去 30 天網站觀察的流量。
  • 積分
    你掛網得到的積分(包含電腦掛網和手機掛網)。

  • 裝置
    裝置的部分可以連結無數個(只要從那台裝置登入就直接連結了)，連結越多，賺的速度越快。

  • 邀請
    這部分就是每個賺錢平台都會有的邀請功能，你每邀請一位朋友，對方每日流量的 10% 會當作你的獎勵。

honeygain如何出金


　　(當honeygain累積到20美元(20000積分)，即可以將累積的金額轉到Paypal平台上。可以參考此網站說明) 　　

honeygain常見問題

  • honeygain只要有網路就可以掛嗎？
    回答：行動網路或者 WiFi都可以。

  • honeygain一天可以掛多少流量？
    回答：根據每個人的網路速度不同而訂，一個裝置最多可以掛到15GB(1500積分)。

  • 有辦法讓收入增加更多嗎？
    回答：有，連結更多裝置同時掛網，官網表示一個IP位址不超過3個是最好的(最多5個)。

  • 如何連結更多裝置？
    回答：只要在欲連結的裝置上開啟網頁(或下載APP)登入帳號即可，一個帳號可以連很多個裝置。

  • 要怎麼關閉Honeygain？
    回答：手機版的直接滑掉。電腦版的到右下角對蜜蜂圖示按右鍵選單中的「Quit」關掉。

  • honeygain的積分會過期嗎？
    回答：會，如果六個月內完全沒有繼續掛網就會過期。

  • 出金方式有哪些？
    回答：目前出金方式只支援Paypal，未來還會陸續開發其餘出金方式。

  • 如果用VPN，honeygain會運作嗎？
    回答：不會，honeygain不支援VPN。
